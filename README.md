# Portfolio Assistant

Portfolio website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Features
- Modern portfolio landing page
- Featured skills, experience, and project sections
- "Ask About Me" assistant without AI API
- Rule-based intent matching and project recommendations

## Run locally
```bash
npm install
npm run dev
```

Open http://localhost:3000

## Main folders
- `src/data`: portfolio content
- `src/lib/assistant`: assistant logic
- `src/components/assistant`: chat UI
